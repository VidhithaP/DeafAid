About Deaf Aid

Deaf Aid is a web application designed to assist individuals with hearing impairments. It uses computer vision and speech recognition to provide real-time information about the user's surroundings.

The application uses the device's camera to detect people and objects, providing a visual description of the environment. It also uses the microphone to transcribe speech into text, which is displayed on the screen.

This application aims to improve communication and accessibility for the deaf and hard-of-hearing community.
